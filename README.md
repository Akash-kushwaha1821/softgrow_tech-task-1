# Iris Dataset EDA App

This project is a simple exploratory data analysis web application built using Python and the Streamlit framework. It analyzes the Iris dataset available in scikit-learn and presents statistical summaries along with visualizations.

The application allows users to preview the dataset, check summary statistics, and explore visualizations such as histograms, box plots, correlation heatmaps, and pairplots. It also highlights key observations about the dataset.

This project demonstrates basic skills in data preprocessing, visualization, and building interactive data science applications.

## Requirements
Python libraries used:
- Streamlit
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

Install them using:

pip install -r requirements.txt

## Run the App

streamlit run app.py
